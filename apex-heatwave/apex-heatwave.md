# Introduction

## Task 1 Setup MySQL Support in Database Tools in OCI

The Oracle Database Development Tools team launched the Database Tools service in OCI providing instance web browser SQL access to to create connections to the MySQL Database Service in OCI. 

1. Menu: Identity & Security>  
    - Create Vault: **MDS-HW**
    - Create Key **MDS-HW**
    - Create Secret **MDS-HW**  Secret is admin login **password** 
2. Menu: Developer>Database Tools
    - Create Private Endpoints **MDS-HW**
3. Menu: Developer>Database Tools
    - Create Connection **MDS-HW**
4. Run SQL Worksheet
